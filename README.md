# Mundo do Devocional

Este é um projeto simples desenvolvido por mim com o objetivo de praticar HTML básico 
enquanto construo algo com significado pessoal.

## O que utilizei

* Estrutura básica de HTML5
* Tags semânticas (head, body, footer)
* Títulos, parágrafos e listas
* Inserção de imagens
* Links externos

## Objetivo

Praticar os primeiros conceitos de desenvolvimento web criando uma página simples sobre devocional, 
unindo aprendizado técnico com algo que faz parte da minha rotina.

## Próximos passos

Aprender CSS para melhorar o visual, organizar melhor o layout e evoluir o projeto aos poucos 
conforme avanço nos estudos.
